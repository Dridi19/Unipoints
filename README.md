### To run this project simply access backend file and run the following commands:

## npm init -y
## npm install express cors mongoose dotenv
## npm install -g nodemon
## nodemon server
## npm i Bcrypt 

### then front file:

## npm install axios
## npm i react-qr-reader
## `npm start`
